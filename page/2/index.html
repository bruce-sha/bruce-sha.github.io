本文主要介绍 Dropbox 作为 Git 私有仓库。你也可以使用其它云存储工具，如 SkyDrive， Google Drive等，或国内的金山快盘，百度云盘，360云盘。对于源代码这些重要资料，我强烈推荐大家使用国外的云产品，百度云曾经丢过我的文件，实在信不过，只作为电影备份盘。
